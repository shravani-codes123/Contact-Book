# 📇 Contact Book

A simple Contact Book application built using HTML, CSS, and Vanilla JavaScript.

## Features

- ✅ Add Contact
- ✅ View All Contacts
- ✅ Edit Contact
- ✅ Update Contact
- ✅ Delete Contact

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Project Structure

```text
ContactBook/
│── index.html
│── style.css
│── script.js
```

## How to Run

1. Clone the repository.
2. Open the project in VS Code.
3. Open `index.html` in your browser.
